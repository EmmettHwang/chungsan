## 🎨 가이드 설계 원칙

### 1. 사용자 중심
- 초보자도 쉽게 따라할 수 있도록
- 단계별 스크린샷은 없지만 명확한 설명
- 각 단계마다 확인 방법 제공

### 2. 복사-붙여넣기 최적화
- 모든 명령어 코드 블록으로 제공
- 한 줄 명령어 제공 (CMD/PowerShell/Bash)
- JSON 예시도 바로 복사 가능

### 3. 문제 해결 중심
- 자주 발생하는 오류 10가지 제공
- 각 오류마다 해결 방법 상세 설명
- "이런 경우에는..." 시나리오 제공

### 4. 실전 위주
- 이론보다 실습
- 5명 팀 프로젝트 시나리오
- curl 명령어로 바로 테스트

### 5. 완벽한 네비게이션
- 각 가이드에서 다른 가이드 링크
- 플로우차트로 가이드 선택 지원
- 학습 경로 추천
