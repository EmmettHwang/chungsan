# 📚 RAG 기본 문서 폴더

이 폴더에 있는 문서들은 백엔드 시작 시 **자동으로 RAG 시스템에 로드**됩니다.

## 📋 사용 방법

### 1️⃣ 문서 추가
이 폴더(`documents/`)에 교재, 교육자료, 가이드 문서 등을 추가하세요.

### 2️⃣ 지원 파일 형식
- **PDF**: `.pdf`
- **Word**: `.docx`, `.doc`
- **텍스트**: `.txt`

### 3️⃣ 파일명 규칙 (권장)
메타데이터를 자동으로 추출하기 위해 다음 형식을 권장합니다:

```
유형_과목명_강사명_날짜.확장자
```

**예시:**
- `교재_바이오헬스기초_홍길동_2024.pdf`
- `가이드_주강사등록_김철수_2024.docx`
- `실습자료_유전자분석_이영희_2024.pdf`

### 4️⃣ 자동 로드
- 백엔드 시작 시 자동으로 이 폴더의 모든 문서를 RAG에 업로드합니다
- 이미 업로드된 문서는 중복 방지 로직이 적용됩니다
- 로드 결과는 백엔드 콘솔에 출력됩니다

## 📊 상태 확인

### 백엔드 콘솔
백엔드 시작 시 다음과 같은 로그를 확인할 수 있습니다:

```
🔄 RAG 시스템 초기화 중...
✅ RAG 시스템 초기화 완료
📚 저장된 문서 수: 0

📚 기본 문서 자동 로드 시작 (8개 파일)
============================================================
✅ 교재_바이오헬스기초_홍길동_2024.pdf: 45개 청크 로드 완료
✅ 가이드_주강사등록_김철수_2024.docx: 12개 청크 로드 완료
...
============================================================
📊 기본 문서 로드 완료: 8개 성공, 0개 실패
📚 현재 총 문서 수: 8
```

### 프론트엔드
1. "시스템 등록" 메뉴 접속
2. "RAG 지식 검색 설정" 섹션에서 상태 확인
3. "상태 새로고침" 버튼 클릭

## 🚨 문제 해결

### 문서가 로드되지 않는 경우
1. 파일 형식 확인 (PDF/DOCX/TXT만 지원)
2. 파일 크기 확인 (50MB 이하 권장)
3. 백엔드 콘솔의 오류 메시지 확인

### 검색 결과가 없는 경우
1. 문서가 제대로 로드되었는지 확인
2. 질문이 문서 내용과 관련 있는지 확인
3. Top-K 값 조정 (시스템 설정에서 변경 가능)

## 💡 팁

### 문서 품질
- 명확하고 구조화된 문서일수록 검색 품질이 향상됩니다
- 목차, 제목, 섹션 구분이 잘 되어 있는 문서를 사용하세요
- 이미지만 있는 PDF는 검색되지 않습니다 (OCR 필요)

### 메타데이터
- 과목명, 강사명 등을 파일명에 포함하면 자동 분류됩니다
- 추후 메타데이터 필터링 기능 추가 예정
