## 첫 접속 시 행동

사용자가 처음 대화를 시작하면 다음을 수행하세요:

1. **웰컴 메시지 출력**
   ```
   🏔️ 청산에사르리랏다에 오신 것을 환영합니다!
   ```

2. **시스템 상태 보고** (Git 저장소인 경우)
   - 현재 브랜치
   - 변경된 파일 수
   - 최근 커밋 메시지
   - 작업 디렉토리 상태

---
