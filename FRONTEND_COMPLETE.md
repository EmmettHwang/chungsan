## 🎓 사용 시나리오

### 시나리오 1: 첫 참여자 추가

1. **브라우저에서 접속**
   ```
   http://localhost:8001
   ```

2. **[참여자 관리] 클릭**

3. **[+ 참여자 추가] 버튼 클릭**

4. **모달에 정보 입력**
   - 이름: 홍길동
   - 역할: Admin (관리자)
   - 기본 수익률: 30% (자동 설정)
   - 연락처: 010-1234-5678
   - 은행: 국민은행
   - 계좌번호: 123-456-789012

5. **[저장] 버튼 클릭**

6. **성공 메시지 확인**
   ```
   ✅ 참여자가 추가되었습니다!
   ```

7. **참여자 목록에서 확인**
   - 코드: HUMAN-001
   - 이름: 홍길동
   - 역할: 관리자 (빨강 뱃지)
   - 수익률: 30%

### 시나리오 2: 프로젝트 생성 및 정산

1. **[프로젝트 관리] 클릭**

2. **[+ 프로젝트 추가] 버튼 클릭**

3. **모달에 정보 입력**
   - 프로젝트명: 테스트 프로젝트
   - 클라이언트: 테스트 고객
   - 총액: 10,000,000원
   - 원가: 3,000,000원
   - 상태: 완료

4. **[저장] 버튼 클릭**

5. **[정산 계산] 페이지로 이동**

6. **프로젝트 선택**
   - 테스트 프로젝트 (테스트 고객)

7. **[정산 계산] 버튼 클릭**

8. **정산 결과 확인**
   - 총 순이익: 7,000,000원
   - 홍길동 (30%): 2,100,000원
